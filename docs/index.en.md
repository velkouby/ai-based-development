# Put coding agents to work inside a verifiable system

Practical principles and engineering practices for moving beyond a conversation with an agent toward an explicit, controlled, and reviewable development process.

The series follows one thread: prepare the repository, choose a level of control proportionate to risk, compile the execution context, know when to stop and how to resume, and produce evidence whose scope is clear.
