# Du principe à la mise en œuvre

Cette série présente une méthode pratique pour intégrer les agents de code dans un système de développement explicite, borné et vérifiable.

Le premier article est une présentation générale du système. Les six suivants sont des approfondissements pratiques qui suivent le même fil conducteur et montrent comment mettre en œuvre chaque couche, dans l'ordre du parcours : repository, niveau de contrôle, exécution, ordre de mission, arrêt et reprise, puis preuve locale.
